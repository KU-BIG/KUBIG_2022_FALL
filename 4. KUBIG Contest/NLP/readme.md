# 금융 뉴스 요약 메일링 서비스

## 1 .프로젝트 소개
**최신 금융 뉴스**를 대상으로 **텍스트 요약 모델**을 활용하여 매일 특정 시간대에 **메일링** 해주는 서비스
- **기간** : 2022.08.01 - 2022.09,01
- **팀원** : 김제성, 김지후, 남정재, 이병주, 이승은
