Cross-Media based Book Recommendation System

추천시스템 2팀(15기 우명진, 16기 임정준/하예은)은 ‘사용자 영화취향을 반영한 크로스미디어 플랫폼 도서 추천 시스템’을 기반으로 영화 및 도서 평점 데이터를 기반으로, 아직 읽지 않은 도서를 사용자에게 추천하는 시스템을 개발하였습니다. 상대적으로 접근성이 용이하고 풍부한 영화 데이터를 활용하여 도서 추천 결과의 예측성을 개선하고, 이를 모델에 반영하였습니다. 이번 프로젝트를 통하여 추천시스템의 기본 개념들을 익히고, 개괄적인 Modeling 및 성능 개선을 넘어, 실제로 모델을 어떻게 Serving하는지까지 간단하게 구현할 수 있었습니다.

Hybrid Recommendation Model로 많이 사용되는 LightFM 모델을 활용하여 모델링을 진행하였으며, 이를 Goorm IDE에서 Flask를 활용하여 실제 Web으로 간단하게 구현하였습니다. 데이터는 Selenium기반으로 Watchapedia의 댓글 및 평점 데이터를 Crawling하여 확보하였으며, 개괄적으로 영화 2200편, 도서 250권, 총 평점 데이터 650,000건을 활용하였습니다.

하단에 보이는 4가지의 사진을 통해서 {{user}}님에게 추천되는 4가지 도서를 안내합니다. '새로고침'을 누르시면 다른 사용자에게 추천되는 다른 도서들도 확인하실 수 있습니다.
